
#  Dashboard d’Analyse des Ventes – Power BI

##  Objectif du projet
Réaliser une analyse complète des performances commerciales d’un dataset e-commerce à l’aide de Power BI afin d’identifier les leviers de rentabilité et les axes d’amélioration.
---
##  Fonctionnalités du dashboard
- Indicateurs clés (KPI) :
  - Chiffre d’affaires total
  - Profit total
  - Quantité vendue
  - Marge (%)
- Analyse géographique :
  - Ventes par région
  - Carte des ventes par État
- Analyse par catégorie :
  - Ventes par catégorie
  - Profit par sous-catégorie
  - Identification des segments non rentables
- Analyse stratégique :
  - Étude de l’impact des remises (Discount) sur la rentabilité
  - Nuage de points (Discount vs Profit) avec ligne de tendance
- Filtres interactifs :
  - Segment
  - Région
  - Catégorie
---

## 🛠 Outils utilisés

- Power BI Desktop
- DAX (mesures personnalisées)
- Nettoyage et transformation de données (Power Query)
- Data Visualisation
